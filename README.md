# proyecto-grupo4-mcdi501

**MCDI501 — Estadística Computacional para la Toma de Decisiones**
Magíster en Ciencia de Datos e Inteligencia Artificial — Grupo 4

## Descripción

Proyecto de análisis estadístico computacional sobre el dataset **Bank Marketing**,
que contiene datos de campañas de marketing directo de una institución bancaria portuguesa.

**Pregunta central:** ¿Es posible predecir si un cliente suscribirá un depósito a plazo
fijo utilizando sus características demográficas, el historial de contactos de campaña
y los indicadores económicos del contexto?

**Fuente del dataset:** [UCI Machine Learning Repository — Bank Marketing](https://archive.ics.uci.edu/dataset/222/bank+marketing)
